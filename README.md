## プロジェクト名
### Movice

## 概要
映画の情報をウォッチリストに追加して、保存することができる。

## 環境構築
1. リポジトリをclone 

- *git clone https://github.com/atsuki-uenishi/movice*

2. cocoapodsをインストール 

- *sudo gem install cocoapods*

3. cocoapodsをセットアップ

- *pod setup*

4. ライブラリのインストール

- *pod install
